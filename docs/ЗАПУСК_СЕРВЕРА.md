# Запуск сервера (если PowerShell выдаёт ошибку выполнения скриптов)

## Вариант 1: Через файл start-crm-server.bat (рекомендуется)

1. Откройте папку проекта `c:\11` в Проводнике.
2. Дважды нажмите на файл **`start-crm-server.bat`**.
3. Откроется окно командной строки (cmd), сервер запустится. Окно не закрывайте.
4. В браузере откройте http://localhost:3000

Чтобы остановить сервер — закройте это окно или нажмите в нём Ctrl+C.

---

## Вариант 2: Через командную строку (cmd), не PowerShell

1. Нажмите **Win + R**, введите **cmd**, нажмите Enter.
2. Выполните:
   ```
   cd c:\11
   npm run dev
   ```
3. Дождитесь сообщения про http://localhost:3000

---

## Вариант 3: Разрешить скрипты в PowerShell (один раз)

Если хотите запускать `npm run dev` именно в PowerShell:

1. Запустите **PowerShell от имени администратора** (правый клик по PowerShell → «Запуск от имени администратора»).
2. Выполните одну команду:
   ```
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```
3. Подтвердите **Y** и Enter.
4. Закройте окно. В обычном PowerShell теперь будет работать `npm run dev`.

После этого в папке `c:\11` можно снова выполнять: `npm run dev`.
